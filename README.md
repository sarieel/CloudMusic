# CloudMusic

10/23 更新路由懒加载，增加歌单页加载时骨架屏效果

- 接口地址：https://github.com/Binaryify/NeteaseCloudMusicApi
- 技术栈：vue3、vue-router、vuex、Element-ui、axios
- 完成功能：发现页、歌单详情页、歌曲播放全屏页
- 亮点：播放器进度条、歌词滚动
